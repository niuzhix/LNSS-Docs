# **欢迎使用 LNSS 聊天系统**

LNSS（本地网络套接字服务器）是一个基于Python和Tkinter构建的局域网聊天应用程序。

## **主要功能**

### *多用户聊天*
支持单个聊天会话中的多个用户。

### *表情支持*
轻松发送表情来表达自己。

### *聊天记录保存*
加密保存您的聊天记录。
